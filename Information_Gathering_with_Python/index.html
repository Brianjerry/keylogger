Information Gathering with Python

Hey there!

Information gathering is the process of gathering information from a target system. It is the first step in any penetration testing or security assessment.

In the first chapter of the Ethical Hacking with Python EBook, we cover the following topics:

  • Extracting domain name information: You will learn to use the WHOIS database and DNS enumeration in Python to extract domain name information. You'll also have a chance to build a subdomain enumeration tool using the requests library.
  • Extracting IP address geolocation: You'll utilize the IPinfo service API to get geolocation from IP addresses for free.
  • Port scanning and enumeration: First, you will build a simple port scanner to familiarize you with the topic. You then dive into a threaded (i.e., faster) port scanner using the sockets library. After that, you will learn to use the Nmap tool in Python to enumerate open ports, services, and even programs running on a particular target system's port.
